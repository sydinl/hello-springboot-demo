# 项目管理功能说明

## 功能概述

项目管理页面提供了完整的项目增删改查功能，包括：

### 1. 项目列表管理
- **分页显示**：支持分页浏览项目列表
- **筛选功能**：按状态、分类筛选项目
- **搜索功能**：按项目名称搜索
- **批量操作**：支持批量删除项目

### 2. 项目创建
- **基本信息**：项目名称、分类、价格、时长
- **状态管理**：启用、禁用、草稿状态
- **图片设置**：项目主图URL
- **详细描述**：项目描述和详情
- **特殊标记**：热门项目、推荐项目标记

### 3. 项目编辑
- **完整编辑**：修改项目的所有属性
- **状态切换**：实时切换项目状态
- **标记管理**：设置/取消热门和推荐标记

### 4. 项目删除
- **单个删除**：删除单个项目
- **批量删除**：选择多个项目批量删除
- **确认机制**：删除前需要确认

## 使用方法

### 访问管理页面
1. 启动应用后访问：`http://localhost:8080/admin/dashboard`
2. 点击左侧导航栏的"项目管理"
3. 进入项目管理界面

### 创建新项目
1. 点击"新建项目"按钮
2. 填写项目信息（标*为必填项）
3. 点击"保存"按钮

### 编辑项目
1. 在项目列表中点击"编辑"按钮
2. 修改项目信息
3. 点击"保存"按钮

### 删除项目
1. **单个删除**：点击项目行的"删除"按钮，确认删除
2. **批量删除**：勾选要删除的项目，点击"批量删除"按钮

### 状态管理
- **热门标记**：点击火焰图标切换热门状态
- **推荐标记**：点击星星图标切换推荐状态
- **项目状态**：在编辑时修改启用/禁用/草稿状态

## API接口

### 项目管理接口
- `GET /api/projects/admin/all` - 获取所有项目（管理用）
- `POST /api/projects/create` - 创建项目
- `PUT /api/projects/update/{id}` - 更新项目
- `DELETE /api/projects/delete/{id}` - 删除项目
- `DELETE /api/projects/batch-delete` - 批量删除项目
- `PUT /api/projects/set-hot/{id}` - 设置热门状态
- `PUT /api/projects/set-recommend/{id}` - 设置推荐状态

### 分类管理接口
- `GET /api/projects/categories` - 获取项目分类列表

## 技术特性

### 前端特性
- **响应式设计**：适配不同屏幕尺寸
- **实时更新**：操作后自动刷新列表
- **用户友好**：直观的操作界面和确认提示
- **数据验证**：前端表单验证

### 后端特性
- **事务管理**：确保数据一致性
- **异常处理**：完善的错误处理机制
- **日志记录**：详细的操作日志
- **权限控制**：需要管理员权限

## 注意事项

1. **权限要求**：所有项目管理操作都需要管理员权限
2. **数据验证**：项目名称、分类、价格为必填项
3. **状态管理**：项目状态影响在前端的显示
4. **图片URL**：需要提供有效的图片URL地址
5. **批量操作**：批量删除操作不可撤销，请谨慎使用

## 故障排除

### 常见问题
1. **无法访问管理页面**：检查是否已登录管理员账户
2. **创建项目失败**：检查必填字段是否完整
3. **图片不显示**：检查图片URL是否有效
4. **操作无响应**：检查网络连接和API接口状态

### 调试方法
1. 查看浏览器控制台错误信息
2. 检查应用日志输出
3. 验证API接口返回状态
4. 确认数据库连接正常
